# real-mart-sales-excel-dashboard

Real Mart Sales Performance Dashboard (Excel)
📌 Project Overview
This project involves a comprehensive analysis of sales data for Real Mart, a retail chain. The goal was to transform raw transactional data into an interactive dashboard to help stakeholders identify top-performing regions, profitable product categories, and sales trends over time.

🛠️ Tools Used
Microsoft Excel: Main tool for analysis.

Power Query: Used for data cleaning and transformation.

Pivot Tables: Used for data aggregation and summarization.

Pivot Charts: Used for visual representation.

Slicers & Timelines: Integrated for interactive filtering.

🧹 Data Cleaning & Preparation
Before building the dashboard, I performed the following steps to ensure data integrity:

Handling Missing Values: Identified and filled null values in "Sales" and "Quantity" columns.

Data Standardization: Corrected inconsistent naming in "Product Categories" and "Region" names.

Feature Engineering: Created new columns for Profit Margin % and Delivery Lead Time to provide deeper insights.

Data Formatting: Ensured all currency and date fields were correctly typed for accurate calculation.

📊 Key Insights & Features
The dashboard provides answers to the following business questions:

Sales Trends: A line chart showing monthly sales growth and seasonal peaks.

Regional Performance: A map or bar chart comparing sales across different territories.

Top 5 Products: Identification of the highest-revenue-generating items.

Customer Segment Analysis: Breaking down sales by "Consumer," "Corporate," and "Home Office" segments.

Profitability Analysis: Highlighting categories with the lowest profit margins that require operational review.

🚀 Interactive Elements
To make the dashboard user-friendly for managers, I added:

Category Slicer: Filter the entire dashboard by clicking a specific product category.

Region Slicer: Compare performance between different geographical areas instantly.

Time Slider: Quickly toggle between Yearly, Quarterly, or Monthly views.

💡 Business Impact
By using this dashboard, the management at Real Mart can:

Optimize Inventory: Stock up on "Top 5" products before peak seasons.

Targeted Marketing: Identify underperforming regions and launch localized promotions.

Cost Control: Focus on improving margins for low-profit product categories.

📂 How to View the Project
Download the Real_Mart_Sales_Analysis.xlsx file from this repository.

Open it in Microsoft Excel 2016 or later.

Click "Enable Content" if prompted (to allow Slicers to work).

Use the Slicers on the left to interact with the data.
